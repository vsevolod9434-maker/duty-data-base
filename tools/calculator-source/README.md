# Источник каталога снабжения для калькулятора

Основной файл для импорта:

```text
tools/calculator-source/supply-catalog.tsv
```

Файл уже объединяет обычные позиции, дополнительные позиции и сборки.

## Колонки

- `key` — стабильный уникальный ключ строки для idempotent seed.
- `kind` — `item` для обычной позиции, `bundle` для сборки.
- `category` — категория для фильтра в калькуляторе.
- `categorySortOrder` — порядок категории.
- `sortOrder` — порядок позиции внутри категории.
- `name` — наименование позиции.
- `contents` — состав сборки; пусто для обычных позиций.
- `traderPrice` — цена сталкерского торговца, если есть.
- `basePrice` — наша цена или себестоимость сборки.
- `generalPrice` — «Общая цена (≥25%)».
- `partnerPrice` — «Цена для сотрудничающих сталкеров (≥20%)».
- `tenantPrice` — «Цена для наших жильцов (≥18%)».
- `note` — примечание.
- `source` / `sourceRow` — служебная информация для сверки импорта.

В интерфейсе калькулятора нужно показывать ровно три цены: `generalPrice`, `partnerPrice`, `tenantPrice`.
Скорректированные цены и доходы в этот файл не включены.
